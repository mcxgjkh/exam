> 专为业余无线电爱好者打造的在线刷题工具，支持 **A/B/C类** 操作证书考试，电脑、手机和手表都能用。

**在线使用：[https://exam.bh6rkw.dpdns.org/](https://exam.bh6rkw.dpdns.org/)**

---

## 💡 这是什么？

这是一个可以直接在浏览器里打开刷题的网站，不需要安装任何软件。

无论你是在家里用电脑，还是在户外用手表，都能随时随地刷题备考。

---

## ✨ 主要功能

### 📝 刷题练习
- A/B/C 三类题库，按顺序、倒序、乱序三种方式刷题
- 做完立刻显示对错，错题自动记下来
- 可以把不会的题目收藏起来反复练

### 📝 模拟考试
- 完全模拟真实考试：题目数量、答题时间、及格线都和官方一样
  - A类：40题 / 40分钟 / 30分及格
  - B类：60题 / 60分钟 / 45分及格
  - C类：90题 / 90分钟 / 70分及格
- 交卷后自动算分，错题自动保存
- 所有考试记录都留着，方便回头看

### 🔍 搜题查漏
- 用题目编号、关键词、甚至拼音首字母都能搜
- 搜到的结果会高亮标出来，点一下就能看到完整题目和答案

### 📱 多端支持
- **电脑**：大屏展示，体验完整
- **手表**：专为小屏优化，字大按钮大，通勤路上也能刷几题

### 💾 数据不丢
- 错题、收藏、考试记录都存在你浏览器的本地记录里，关掉网页再打开还在
- 支持云端同步（可选），换了设备也能接着练
