欢迎使用"业余无线电考试刷题模拟系统"（以下简称"本系统"）。本系统由 BH6RKW 开发并维护，致力于为业余无线电爱好者提供免费、高效的题库练习与模拟考试环境。

请您在使用本系统前仔细阅读本协议全部条款，特别是关于账号注册、数据存储（本地与云端）、第三方服务及免责声明的内容。当您在本系统主页勾选"我已阅读并同意"并继续使用时，即视为您已理解并完全接受本协议的全部条款。如您不同意本协议的任何内容，请立即停止使用本系统并清除相关本地与云端数据。

本系统根据 AGPL-3.0 协议在 GitHub 开放源代码。我们欢迎您为本项目做出贡献（提交 Issue、Pull Request 等），亦可在遵守开源协议的前提下进行二次创作、修改、再分发。

核心原则：本系统绝不收集或出售您的个人身份信息。本地练习数据（错题、收藏、待做进度、历史记录）仅存储在您的浏览器中，完全由您控制。若您选择使用云同步功能，相关数据将加密传输至 Supabase 云端存储，您可随时在个人中心删除云端数据或注销账号。

一、服务功能概述

本系统提供 A/B/C 类业余无线电题库的以下功能：

  • 模拟考试：按官方标准计时、计分，模拟真实考试环境。
  • 题目学习：支持顺序、倒序、乱序三种模式浏览练习全部题目。
  • 待做续练：自动保存学习进度，下次进入同模式可无缝继续。
  • 错题巩固：自动收集错题，支持专项反复练习。
  • 收藏题目：手动收藏重点题目，集中复习。
  • 题目查询：按题目编号或关键词搜索题库。
  • 考试历史：记录每次模拟考试的成绩、用时、对错情况。
  • 云同步（可选）：注册账号后可将错题、收藏、历史记录同步至云端，跨设备使用。

二、本地数据存储说明

无论您是否注册账号，本系统均会在您的浏览器本地存储以下数据以实现核心功能：

  • 错题本：记录您答错的题目 ID，用于错题专项练习。
  • 收藏夹：您主动收藏的题目 ID，便于集中复习。
  • 待做会话：学习模式下的答题进度（题目列表、已作答记录、当前索引、选项随机顺序），下次进入同模式可继续。
  • 考试历史：每次模拟考试的成绩、用时、类别等记录。
  • 主题偏好：您选择的日间/夜间模式。
  • 选项随机顺序：每题选项的随机排列，确保练习多样性，不影响标准答案判断。

存储技术：以上数据均通过浏览器 LocalStorage 存储在您的设备上，遵循浏览器同源策略，仅限 exam.bh6rkw.dpdns.org 域名访问。除您主动使用云同步功能外，这些数据不会以任何形式发送至任何服务器。

存储期限：数据将长期保留，除非您主动清除浏览器数据或使用系统内提供的清除功能。您可以在浏览器设置中随时删除本网站的所有本地存储数据。

三、账号系统与云同步（可选功能）

本系统提供完全可选的账号注册与云同步功能。您无需注册账号即可使用全部刷题功能。

3.1 账号注册

  • 注册需提供电子邮箱和密码（至少 8 位，需包含字母和数字）。用户名可选填（10 位以内），仅用于界面显示。
  • 注册后系统将向您的邮箱发送确认链接，点击确认后账号激活。未确认的账号可能无法使用完整功能。
  • 您应对账号密码妥善保管，因密码泄露导致的数据问题，开发者不承担责任。
  • 您可通过"忘记密码"功能重置密码，系统将向注册邮箱发送重置链接。

3.2 云同步机制

  • 登录后，您可手动点击"同步"按钮将本地错题、收藏、历史记录上传至云端，或从云端拉取数据合并到本地。
  • 同步采用智能合并策略：本地与云端数据按题目 ID 去重合并，历史记录按时间戳去重，不会重复计数。
  • 练习过程中，错题和收藏记录会实时上传至云端（静默进行，不影响使用体验）。
  • 云同步功能依赖网络连接，网络异常时数据暂存本地，待网络恢复后可手动同步。

3.3 云端数据存储

  • 云端数据存储在 Supabase（东亚）提供的数据库中，包括：用户 ID、题目 ID、考试类别、对错标记、收藏标记、答题时间、考试记录等。
  • 我们不会在云端存储您的明文密码。密码由 Supabase 使用行业标准哈希算法（bcrypt）加密存储。
  • 云端数据仅通过 HTTPS 加密传输。

3.4 注销账号与删除云端数据

  • 您可随时通过系统内个人中心页面删除全部云端数据并注销账号，操作不可逆。
  • 注销后，您的邮箱、用户名、所有练习记录将从 Supabase 数据库中永久删除，无法恢复。
  • 本地浏览器数据不受注销影响，您需另行清除浏览器本地存储，详见浏览器官方文档。

四、信息收集与隐私保护

本系统遵循数据最小化原则，具体说明如下：

4.1 我们不收集的信息

  • 不收集姓名、电话、身份证号、地理位置等个人身份信息。
  • 不使用第三方统计或分析工具（如 Google Analytics、百度统计）。
  • 不设置跨站追踪，不投放广告，不出售或共享任何用户数据。
  • 题库文件为纯静态资源，不含任何追踪代码。

4.2 使用账号功能时会涉及的信息

  • 邮箱地址：用于账号注册、登录验证和密码重置，由 Supabase 存储。
  • 密码：由 Supabase 使用 bcrypt 哈希加密存储，开发者无法获知您的明文密码。
  • 用户名（选填）：仅用于界面展示，非必填。
  • 练习数据：错题 ID、收藏 ID、考试记录等，用于云端同步。
  • 认证令牌（Token）：登录后 Supabase 会在浏览器 LocalStorage 中存储访问令牌和刷新令牌，用于维持登录状态。退出登录或注销账号时清除。

4.3 第三方服务说明

本系统依赖以下第三方服务，其隐私政策与本系统独立，建议您同步了解：

  • Supabase（supabase.com）：提供账号认证和云端数据库服务。您的邮箱、哈希密码及练习数据存储于 Supabase 服务器。详见 Supabase 隐私政策。
  • Cloudflare Turnstile（cloudflare.com）：在登录、注册、密码重置时进行人机验证，用于防止自动化攻击和滥用。Turnstile 可能收集浏览器特征、设备信息等用于判断是否为真人操作。详见 Cloudflare 隐私政策。
  • jsDelivr CDN（jsdelivr.net）：用于加载 Supabase 客户端库等静态资源，可能记录请求日志。

开发者不对上述第三方服务的数据处理行为承担责任，但我们已选择信誉良好的服务商并仅使用必要功能。

五、用户权利与数据管理

您对本系统产生的所有数据享有完全的控制权：

  • 查看本地数据：通过浏览器开发者工具（F12 → 应用程序 → 本地存储）可查看所有本地存储数据，格式为 JSON，可自行备份导出。
  • 删除本地数据：通过浏览器设置清除本网站数据，或使用系统内功能移除错题/取消收藏。考试历史可在主页历史记录区域批量删除。
  • 管理云端数据：登录后通过个人中心查看云端存储内容，可手动删除特定记录或一键清除全部云端数据并注销账号。
  • 数据可携带：本地数据以 JSON 格式存储，您可随时通过浏览器开发者工具导出备份。

系统版本更新不会覆盖或删除您原有的本地数据（除非题库结构发生重大不兼容变更，届时系统会在主页弹窗提示）。
